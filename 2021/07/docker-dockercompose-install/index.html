Docker Dockercompose Install

  2021-07-22  评论 字数统计: 433字 阅读时长: 1分

安装docker

0.更新软件包，确保服务器的软件包是最新的

sudo yum update -y

1.安装必要依赖

sudo yum install -y yum-utils device-mapper-persistent-data lvm2

2.添加软件源信息

sudo yum-config-manager --add-repo http://mirrors.aliyun.com/docker-ce/linux/centos/docker-ce.repo

3.更新 yum 缓存

sudo yum makecache fast

4.安装 Docker

sudo yum install docker-ce docker-ce-cli containerd.io

5.启动 Docker 后台服务

sudo systemctl start docker

6.允许当前用户直接运行 docker 命令

需要将当前用户加入 docker 用户组，这样每次运行 docker 命令的时候，就不需要加 sudo了

sudo usermod -aG docker your_name   #设置成功之后需要重新登录才会生效

7.镜像加速

1.使用阿里云镜像加速

进入阿里云容器镜像服务，在左侧选择镜像加速器，可以看到加速器地址，按照操作文档进行配置

2.使用网易云镜像加速

新建 daemon.json 文件

sudo vim /etc/docker/daemon.json

将下面的配置粘贴进去保存即可

{
  "registry-mirrors": ["http://hub-mirror.c.163.com"]
}

重载配置文件

sudo systemctl daemon-reload

重启docker服务

sudo systemctl docker

安装docker-compose

0.运行以下命令以下载 Docker Compose 的当前稳定版本

#将docker-compose安装到/usr/local/bin目录下
sudo curl -L "https://github.com/docker/compose/releases/download/1.24.1/docker-compose-$(uname -s)-$(uname -m)" -o /usr/local/bin/docker-compose

1.给予docker-compose可执行权限

sudo chmod +x /usr/local/bin/docker-compose

2.创建软链

sudo ln -s /usr/local/bin/docker-compose /usr/bin/docker-compose

3.查看docker-compose是否安装成功

docker-compose --version
